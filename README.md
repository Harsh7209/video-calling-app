# Video Calling App

## Overview
This video calling app allows users to connect and communicate in real-time through video calls. Built using modern web technologies, it offers a seamless experience for users to interact with each other.

## Features
- **Real-time video communication**: Users can initiate video calls with a single click.
- **Multi-user support**: Engage in group calls with multiple participants.
- **Chat functionality**: Text chat support during video calls.
- **User-friendly interface**: Simple and intuitive UI for easy navigation.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript, React.js
- **Backend**: Node.js, Express.js
- **WebRTC**: For real-time communication
- **Socket.io**: For real-time event handling

## Getting Started
### Prerequisites
- Node.js
- npm or Yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Harsh7209/video-calling-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd video-calling-app
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running the Application
- Start the server:
   ```bash
   npm start
   ```
- Open your browser and navigate to `http://localhost:3000`.

## Usage
- To initiate a video call, click on the "Start Call" button and share the generated link with your friends.
- You can also join existing calls with the provided link.
